這是我的個人 Alacritty 終端機設定檔，旨在提供一個美觀、高效且易於閱讀的終端環境。

- **配色方案**: 使用受歡迎的 **Catppuccin Mocha** 風格，色彩柔和且對眼睛友好。
- **透明效果**: 啟動了 90% 的透明度，並包含背景模糊（Blur）效果，提供現代化的視覺體驗。

  - 10,000 行的歷史滾動記錄。
  - 預設 Shell 設定為 `zsh` 並以 login shell 模式啟動。
  - 游標樣式改為閃爍的 `Beam` (I字型)，更符合現代輸入習慣。

## 🛠️ 安裝與使用

### 1. 必要條件

在套用此設定前，請確保您的系統已安裝以下組件：

- **Alacritty**: 現代化的高效能 GPU 加速終端機。
- **JetBrainsMono Nerd Font**: 為了正確顯示圖示與連字，建議安裝此字體。
  - 可以從 [Nerd Fonts 官網](https://www.nerdfonts.com/font-downloads) 下載。
- **zsh**: 本設定檔預設使用 `/usr/bin/zsh`。

### 2. 套用設定

將 `alacritty.toml` 放置到您的 Alacritty 設定路徑：

- **Linux/macOS**: `~/.config/alacritty/alacritty.toml`
- **Windows**: `%APPDATA%\alacritty\alacritty.toml`

您可以直接連結或複製此檔案：

```bash
mkdir -p ~/.config/alacritty
cp alacritty.toml ~/.config/alacritty/alacritty.toml
```
